# Trabajo de Fin de Máster (TFM)

## Descripción
Este repositorio contiene el código y los recursos asociados al Trabajo de Fin de Máster titulado:


El objetivo de este proyecto es analizar el comportamiento de espumas en blancos de ICF directos con dopantes de medio Z.

Input ARWEN: Código para obtener la geometría de la espuma y el perfil del haz láser.
Extracción datos: Análisis y procesado archivos .h5.
Calculos: Obtención de análisis modal, gradientes, tasa de crecimiento RTI.
